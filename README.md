# PaczkoTex
Projekt zajmujący się optymalizacją trasy dostaw dla firmy kurierskiej na terenie miasta Zielona Góra.

### Zasady
- Każda część projektu ma znajdować się w katalogu z adekwatną nazwą - jeśli jeszcze nie istnieje takowy, należy go stworzyć.
- Program, ktory wymaga dodatkowych plików do uruchomienia, należy trzymać w oddzielnym katalogu wewnątrz katalogu kategoryzującego
- Każdy wstawiony plik z kodem, powinien posiadać w komplecie plik .md lub .txt z przynajmniej krótkim, formalnym lub nie, opisem co robi oraz jak go używać +-
